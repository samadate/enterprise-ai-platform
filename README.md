# Enterprise AI Platform

An enterprise-grade AI backend platform built with Java and Spring Boot, designed to demonstrate modern AI integration patterns, Retrieval-Augmented Generation (RAG), provider abstraction, distributed systems, and cloud-native architecture.

> 🚧 Project Status: In Active Development

## Vision

Build a production-inspired AI platform that supports multiple LLM providers, enterprise knowledge retrieval, and scalable backend architecture while following software engineering best practices.

## Planned Tech Stack

- Java 21
- Spring Boot
- Ollama
- PostgreSQL
- pgvector
- Apache Kafka
- Redis
- Docker Compose
- Prometheus
- Grafana
- Angular
